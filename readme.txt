Adds the official Tweet Button from Twitter.com. 

== Description ==

This plugin adds the official [Tweet Button](http://blog.twitter.com/2010/08/pushing-our-tweet-button.html) from Twitter to your Pages, Posts or Home Page. It lets your users share links directly from the page they're on. When they click on the Tweet Button, a Tweet box will appear pre-populated with a shortened link that points to the item that they're sharing.

This plugin is very customizable via 'rsh-Tweet Button' under the 'Settings' menu in WordPress Admin. Change the look and feel, pozition and where to seen of the button.

== Installation ==

1. Unzip `rsh-tweet-button.zip`
2. Upload `rsh-tweet-button` directory to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Set up the plugin in 'Settings' menu in WordPress
